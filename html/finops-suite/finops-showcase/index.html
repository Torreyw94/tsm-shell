📖 HOW TO USE THIS SHOWCASE
STEP 1 · CHOOSE MODE
Use SAMPLE for pre-loaded FinOps docs or ACTUAL to upload a real client file.
STEP 2 · SELECT OR UPLOAD
Click a sample card to activate it. For actual docs, drag-drop or click the upload zone.
STEP 3 · PROCESS
Hit ▶ PROCESS. The node chain lights up: INGEST → PARSE → AI → BNCA → REPORT.
STEP 4 · READ OUTPUT
AI returns Summary, Findings, Risk, Flags, and BNCA. KPI cards populate below.
STEP 5 · SAVE REPORT
Hit 💾 SAVE REPORT to store in FinOps strategist memory for later recall.
STEP 6 · PUSH TO STRATEGIST
Hit → PUSH to send the report into FinOps Main Strategist for cross-node action.

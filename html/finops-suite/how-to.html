COREStaff Accountant Copilot
Your unified command layer. Select a workflow lane, optionally paste raw context (invoices, aging data, notes), and the copilot runs all six FinOps lanes simultaneously — returning a ranked action list with owner assignments and a controller-ready summary.
HOW TO USE
01
Choose your workflow lane
Use the dropdown: AP Aging / Vendor Invoices, AR Ledger, 1099 / Tax, Bank Reconciliation, Month-End Close, or Compliance Audit.
02
Paste context (optional but powerful)
Drop raw invoice notes, aging bucket data, reconciliation gaps, or audit findings into the text area. The AI uses this to personalize the output.
03
Click "Run Staff Accountant Copilot"
Calls POST /api/finops/copilot. Returns priority_rank[], combined_bnca, controller_note, and confidence score.
04
Read the ranked output
Actions sorted by impact. Status badges: ACTION REQUIRED WATCH OK
05
Push to controller
Click "Controller Action Plan" to generate a controller-ready summary note from the same session data.
PRO TIP: Paste multiple invoice lines or a CSV-style aging table directly into the context box. The copilot will extract vendor names, amounts, and aging buckets automatically.
LANE · APAP Aging / Vendor Invoices

01

02
Paste aging data

03
Review output fields
issue = what's wrong · impact = close risk · owner = Staff Accountant or Controller · status = action level

Workflow param
"AP Aging"

Vendor · Invoice # · Amount · Bucket
Key output fields
priority_rank · payment_timing · close_risk

PRO TIP: Include "close date: [date]" in your context. The copilot will calculate days-to-close pressure and escalate anything in the 60+ bucket automatically.

LANE · STRATEGYStrategist BNCA
BNCA = Best Next Closing Action. Synthesizes all open lanes into a single prioritized narrative for the controller. Converts parallel FinOps threads into one staff-accountant workflow and a controller-ready action plan.
HOW TO RUN
01
Open Main Strategist
Navigate to /html/finops-suite/finops-main-strategist/index.html or click "Open FinOps Strategist" from Copilot.
02
Load prior lane outputs
Paste or reference output from AP, Compliance, and Zero Trust lanes. The strategist synthesizes across all three.
03
Generate BNCA narrative
Returns combined_bnca: a plain-English closing sequence with ranked next actions and owner assignments.
INPUT
AP + Compliance + Zero Trust lane results
OUTPUT
combined_bnca · controller_note · confidence %
AUDIENCE
Controller · CFO · Audit partner
PRO TIP: Use Strategist BNCA at the end of every close cycle. The controller_note field is designed to be copy-pasted directly into your close memo.
APIMulti-Chain Reports
Runs multiple workflow lanes in a single API call. Returns a consolidated report across all selected workflows — ideal for month-end close packages.
HOW TO RUN
01
Call the multi-report endpoint
POST to /api/finops/multi-report with a workflows array.
02
Specify workflows array
Example: {"workflows":["AP Aging","Compliance Shield","Zero Trust"]}
03
Receive combined output
Returns one consolidated result per workflow plus a cross-lane priority summary.
AVAILABLE WORKFLOW COMBINATIONS
Close Package
AP Aging + Bank Rec + Month-End
Risk Package
Compliance Shield + Zero Trust
Tax Package
1099 Tracker + AR Ledger
Full Suite
All 4 workflows simultaneously
PRO TIP: Schedule the Full Suite call nightly during close week. The multi-chain output gives your controller a live dashboard without manual re-runs.
ADVANCEDGenerating Custom Reports
Any lane can be extended into a custom report by shaping the context you pass. The AI interprets structured context and returns structured JSON you can pipe into any reporting tool.
CUSTOM REPORT TEMPLATE
01
Choose your workflow lane
Pick the lane that matches your report scope (AP, AR, Tax, Compliance, Close, Zero Trust).
02
Structure your context block
Lead with: REPORT TYPE: [name] then data rows then FOCUS: [what you want highlighted]
03
Request specific output fields
End with: OUTPUT: summary, ranked_actions, owner_assignments, deadline — the AI will honor these field names in its JSON response.
04
Parse and export
The response is clean JSON. Pipe it to Excel, Google Sheets, or your GL system. Use priority_rank[] as your report rows.
  • Vendor aging report → AP Aging lane + paste aging table as context
  • 1099 readiness checklist → 1099 Tracker lane + paste vendor W-9 status
  • Month-end close sign-off → Strategist BNCA after all lanes complete
  • Audit trail report → Zero Trust lane + paste transaction log
  • AR collection priority → AR Ledger lane + paste customer aging
  • Bank rec gap report → Bank Rec lane + paste unmatched items
PRO TIP: To generate a PDF-ready report, run the copilot, copy the priority_rank and controller_note fields, and paste into the Doc Showcase — it formats them into a print-ready layout automatically.
CONTROLLERController Action Plan
A one-click summary layer that converts any copilot session into a controller-ready memo. Includes all ranked actions, owner assignments, deadlines, and an executive BNCA narrative.
HOW TO USE
01
Run any copilot workflow first
Complete at least one lane. The Controller Action Plan synthesizes the current session output.
02
Click "Controller Action Plan"
Available in the sticky footer bar at the bottom of the Copilot UI.
03
Review and share
Fields: controller_note · priority_rank · combined_bnca · confidence · ts
04
Export to Doc Showcase
Click "Open Doc Showcase" to render as a formatted, shareable document.
BEST FOR
Month-end close sign-off memos
AUDIENCE
Controller · CFO · External auditor
FORMAT
JSON → Doc Showcase → PDF-ready
PRO TIP: Confidence score below 70 = gather more context before sharing with the controller. Above 85 = ready to sign off.

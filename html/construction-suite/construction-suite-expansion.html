TSM Construction Suite — Overview
A complete field-to-executive construction management platform. All modules are interconnected and AI-powered via Groq's ultra-fast inference engine.
■
FieldOps Daily (Module 02)
Geospatial jobsite capture, daily reports, crew logs, safety checklists, and offline-first photo uploads with auto-geotags.
■
PlanRoom Pro (Module 03)
Enterprise blueprint viewer with OCR text extraction, markup tools, version control, and RFI linking directly to drawing callouts.
■
Procurement (Module 04)
Real-time inventory tracking, PO management, supplier order status, barcode/QR scanning, and AI-powered reorder recommendations.
■
RFI / Change Orders (Module 05)
Full RFI lifecycle management, change order creation with budget linkage, AI response drafting, and approval chain visualization.
■
Scheduling / Gantt (Module 06)
Interactive Gantt chart with critical path highlighting, baseline vs actual tracking, crew assignment, and AI schedule recovery analysis.
■
Groq AI (All Modules)
Ultra-fast LLM assistant wired into every module. Draft RFI responses, analyze schedules, optimize procurement, and summarize field reports in seconds.
► Quick Start
1. Enter your Groq API key (click GROQ AI in the top bar)
2. Select your project in FieldOps Daily and enter the jobsite address to activate the map
3. Load a drawing in PlanRoom Pro and run OCR Scan
4. Review Procurement alerts and run AI Reorder Analysis
5. Check your Gantt for critical path delays and ask AI for recovery options
FieldOps Daily — Module 02
Capture all jobsite data daily — from crew counts to geospatial coordinates — with offline-first reliability.
Geospatial Map Fly-In
  • Select your project from the dropdown at the top of the sidebar. This sets the project context for the daily report.
  • Type the jobsite address into the Jobsite Address field. Any real-world address will work — street address, intersection, or city/state.
  • Press Enter or click → MAP. The map animates with a smooth fly-in to the exact location. A GPS marker pins the site and coordinates are locked.
  • All field photos uploaded in Section 05 are automatically geotagged with these locked coordinates for audit-ready documentation.
► GPS Lock
Once a site address is geocoded, the GPS coordinates are stored and applied to all reports, photos, and submissions for that project date. The green "GPS LOCKED" badge confirms the lock.
Daily Report Workflow
  • Fill in Daily Summary — percent of planned work complete and superintendent notes.
  • Add crew entries — trade, crew size, hours. Click + Add Crew for additional trades.
  • Complete the Safety Checklist — toolbox talk, PPE verification, incident log.
  • Click Save Local to cache offline (IndexedDB), or Submit Report to send to PM with PDF generation.
PlanRoom Pro + OCR — Module 03
Enterprise blueprint management with AI-powered OCR text extraction directly on construction drawings.
Blueprint OCR Viewer
  • Select a drawing from the folder tree or file list in the left sidebar. The blueprint renders instantly in the main viewer.
  • Click OCR SCAN (cyan button, top-right of toolbar). The engine scans the blueprint in real-time, progressing through 10 detection stages.
  • Bounding boxes appear on the canvas highlighting every detected text region — title blocks, room labels, dimensions, grid references, and callouts.
  • The OCR panel (right sidebar) shows confidence score, element count, extracted metadata fields, and raw text output — fully searchable and copyable.
  • Extracted metadata (sheet number, scale, revision, occupancy type) auto-populates the Info panel for instant document classification.
Markup Tools

Use the toolbar to annotate drawings: Pen for freehand, Rect for area callouts, Text for notes, Measure for dimensional verification. Apply stamps (Approve / Revise / Void) with a single click. Use Issue Pin to place numbered flags that link to RFIs in the side panel.

⚠ Version Control
Always upload new revisions as new versions — never overwrite. The Versions tab in the right panel shows the full revision history and supports side-by-side comparison with "Compare Versions."
Procurement & Materials — Module 04
Real-time material inventory, supplier PO tracking, and AI-powered reorder intelligence.
Inventory Management
  • The inventory table shows all materials with on-hand quantities, reorder points, visual stock bars, and supplier lead times.
  • Red rows are critical (below reorder point). Amber rows are warnings (<30 day supply). Green rows are healthy stock.
  • Click Order Now on any item to create a draft PO linked to that material and its preferred supplier.
  • Use the QR/BARCODE SCAN button to simulate field scanning — in production, this connects to a mobile device camera for real-time stock updates.
Groq AI Procurement Analysis

Click AI Reorder Analysis to have Groq analyze your current stock levels against upcoming schedule milestones and generate a prioritized procurement action list. The AI considers lead times, budget constraints, and critical path dependencies.

RFI & Change Orders — Module 05
Full lifecycle management of Requests for Information and Change Orders with AI-assisted drafting.
RFI Workflow
  • Click + New RFI to create a new request. Fill in subject, description, discipline, drawing reference, and assign to the responsible party (architect, engineer, owner).
  • The Approval Chain shows the review flow: Submitted → Under Review → Responded → Closed. Each step shows the responsible party and timestamp.
  • Click AI Draft to have Groq write a professional RFI response based on the subject and issue description. Edit as needed before sending.
  • Link an RFI to a specific drawing in PlanRoom Pro using the Issue Pin tool — the RFI appears in the drawing's RFI panel automatically.
Change Order Creation

Create COs from the right pane. Link to impacted drawings, attach cost breakdowns, and connect to the FinOps budget module. Approval chain tracks: Created → PM Review → Owner Approval → Executed. Use the AI Analysis button to assess schedule and budget impact before submitting.

Scheduling / Gantt — Module 06
Interactive project schedule with critical path visualization and AI-powered recovery planning.
Reading the Gantt

Each task row shows the planned bar (darker) vs. the actual progress bar (brighter). Green bars are on track. Amber bars are delayed but recoverable. Red bars are on the critical path — delays here impact the end date directly.

AI Schedule Recovery
  • Click AI Schedule Analysis in the metrics bar. Groq analyzes the current schedule, identifies the critical path, and suggests acceleration options.
  • The AI considers crew availability, material lead times, weather risk, and subcontractor constraints when generating recommendations.
  • Use Week / Month / Quarter view toggles to zoom the Gantt for different planning horizons.
Groq AI Integration
Ultra-fast AI inference wired into every module. Powered by Groq's LPU hardware — responses typically arrive in under 2 seconds.
Setup — API Key
  • Click the ⚙ GROQ AI button in the top navigation bar.
  • Enter your Groq API key (get one free at console.groq.com). The key is stored only in your browser session — never transmitted except to Groq's API directly.
  • The status badge changes to ACTIVE once the key is validated. All AI buttons across all modules are now live.
AI Capabilities per Module

Procurement: Reorder recommendations, supplier risk assessment, budget impact.

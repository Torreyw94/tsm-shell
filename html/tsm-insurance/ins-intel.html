GUIDE
HOW TO USE THE SUITE
The Insurance Intelligence Suite is designed to flow from intake to resolution. Here's how each category fits together.
01
STARTING POINT
Open with Ins Main Strategist
This is your AI copilot for the entire suite. Use it to orient on any topic — claims queues, compliance status, fraud flags — before diving into individual apps.
Ask it to summarize today's claims priority
Request a compliance risk snapshot before a meeting
02
CLAIMS
Triage before you investigate
Run all new claims through the Triage Engine first. It scores intake priority and routes to the dashboard. Only escalate to Fraud Signal if a red flag fires.
Set triage thresholds before shift start
Use FNOL Intake for same-day loss events
Dashboard shows adjuster workload in real time
03
FRAUD
Signal first, SIU second
The Fraud Signal Analyzer runs pattern detection. Only use the SIU Referral Console when the Analyzer returns a high-confidence flag — it packages the referral package for investigators.
Review signal scores weekly even on low-priority claims
SIU Console auto-generates the referral narrative
04
COMPLIANCE
Shield runs continuously
Compliance Shield monitors regulatory exposure in the background. Check Audit Prep before any DOI exam. Use Reserve Adequacy monthly for IBNR review alongside your actuary.
Export Shield reports for board packets
Audit Prep generates evidence checklists automatically
05
CLIENT-FACING
Client 360 before every touchpoint
Pull up Client 360 before calls, renewals, or claims conversations. AZ Insurance and AZ Life are for Arizona market client presentations and policy quoting.
360 shows all policies, claims, and payment history
Use the Pitch Deck for new business meetings
06
INTERNAL OPS
Underwriting and Agents run in parallel
Agents Intelligence tracks producer performance and licensing. Underwriting Desk manages risk appetite and referrals. Both feed back into the Strategist for operational intelligence.
Agents dashboard auto-flags expired licenses
UW Desk connects appetite matrix to triage output

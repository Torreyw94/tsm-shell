# Internal Audit Findings Report
Audit Area: Procurement Process
Date: March 2026

Findings:
1. Missing approvals for purchases greater than $10,000.
2. Duplicate vendor entries in accounting system.
3. Inconsistent documentation retention.
4. Manual approval trail not consistently preserved.
5. Vendor master cleanup required.

Risk Level: Moderate

Recommendations:
- Implement automated approval workflow.
- Run vendor master cleanup.
- Enforce document retention policy.
- Require controller review for high-dollar purchases.
